1.什么是方法
 方法就是定义在类中的具有特定功能的一段独立小程序，用来完成某个功能操作。在某些语言中方法也称为函数或者过程。
 特殊的 main 方法，该方法是程序的入口，由虚拟机来调用执行。
 在日常的开发中，不可能把程序的所有功能都放到 main 方法中，我们需要定义其它的方法来完成指定的功能，需要完成该功能时只需要调用方法即可。
2.定义方法
    [修饰符] 返回值类型 方法名(参数数据类型 参数名1，参数数据类型 参数名2，……) {

        // 执行语句;
        return 返回值；
    }

    1.方法可以有0到多个修饰符
    2.方法要定义返回值类型
    3.方法名(标识符-程序员自己起的名字)
        标识符：
          1.有 字母 数字 下划线 美元符号组成
          2.不能以数字开头
        命名规范采用小驼峰命名 hello、sayHello
    4.方法参数
      一个方法可以有0到多个参数,多个参数用逗号分隔
      每个参数要明确数据类型
      参数名（标识符）使用小驼峰命名法
    5. { } 内是是方法的内容
    6.方法的返回值
       return 返回值
       方法的返回值，要和方法定义的返回值类型保持一致
       方法可以没有返回值？？？？
    注意：
      如果方法没有返回值，方法的返回值类型使用 void
      如果方法没有返回值，return 语句可以省略
      如果方法有返回值，return语句不能省略
      return 语句一定是在方法的最后面

3.调用方法
   方法名(); // 调用不带参数，且没有返回值的方法
   方法名(参数1,参数2);// 调用带参数，且没有返回值的方法
   数据类型 var = 方法名(参数1,参数2); // 调用有返回值的方法，可以使用一个变量接收方法的返回值

   方法可以被多次调用
   一个方法中可以调用另一个方法
   方法只定义，不调用是不会被执行的

4.方法的重载

  概念
     在同一个类中，如果出现一组方法名相同，参数类型或参数个数、参数顺序不同。
     这样一组同名的方法称为 方法重载

  返回值类型不同，不构成方法的重载。
  形参的名称不同，不构成方法的重载。
  方法修饰符不同，不构成方法的重载。

  注意：重载的方法，实际是完全不同的方法！只是方法名称相同而已！

  优点：
  方法重载通常用来命名一组功能相似的方法，这样做减少了方法名的数量，避免了名字空间的污染，对于程序的可读性有很大的好处。
