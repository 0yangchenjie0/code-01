1.static
  可以修饰 成员变量 --->       静态变量（类变量）final static 修饰后成为静态常量
          成员方法 --->      静态方法(类方法) 封装工具类（Arrays）
          代码块  ---->      静态代码块 类的初始化
  访问方式：
    类名.静态变量
    类名.静态方法
    静态代码块随着类的加载会执行，并且只执行一次
  访问规则：
    静态的可以访问静态的
    非静态的可以访问静态的
2.代码块
    静态代码块 static {}
    构造代码块 {}
  静态代码块 --> 构造代码块 ---> 构造方法

3.包
  package 声明当前类所在的包  com.bjpowernode
  import 导入类
  如果同时需要使用不同包下的同名类，不同都用 import导入
  建议：用的多的类使用 import 导入，用的少使用 包名.类名 方式引入
