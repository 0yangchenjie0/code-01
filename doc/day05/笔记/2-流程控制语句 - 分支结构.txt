1.程序的三种结构
  1）顺序结构
    程序按程序的编写顺序执行
  2）分支结构（选择结构）
    if 选择结构
        if 单选结构
            if(条件表达式) {
               // 当“条件表达式”的结果为true时，则执行此处的代码;
            }
        if 双选结构
            if(条件表达式) {
               // 当“条件表达式”的结果为true时，则执行此处的代码;
            } else {
              // 当“条件表达式”的结果为false时，则执行此处的代码;
            }
        if 多选结构
            if(条件表达式1) {
               // 当“条件表达式1”的结果为true时，则执行此处的代码;
            }
            else if(条件表达式2) {
              // 当“条件表达式2”的结果为true时，则执行此处的代码;
            }
            else if(条件表达式3) {
              // 当“条件表达式3”的结果为true时，则执行此处的代码;
            }
            ....
            else {
              // 当上面的表达式都不成立，执行
            }

       总结：
         if 语句是必须
         else if 可以有 0 到多个
         else 可以有 0个或1个
          1、 如果if选择结构只需执行一条语句时，那么可以省略{}；如果需要执行多条语句，那么就是不能省略{}。为了提高代码的易读性，建议不要省略{}。
          2、 {}中的代码语句也称为代码块，在代码块定义的常量或变量的作用域仅限于代码块中，在代码块之外不能被使用。
          3、 能用if多选择结构就不要用if单选择结构，因为if多选择结构的效率要高于if单选择结构。
          4、 在if选择结构中还可以嵌套别的选择结构或循环结构。
    switch 分支
       switch(表达式) {
         case: 常量1 {
            // 语句序列
            [break;]
         }
         case: 常量2 {
             // 语句序列
             [break;]
         }
          [default:]
             默认语句;
             [break;]
       }
