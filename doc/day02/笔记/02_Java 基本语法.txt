1.注释
  注释是 程序员 在 源代码中 添加的对代码功能解释说明的文字，是用来帮助程序员理解程序的（给人看的）。
  三种类型的注释：
	  单行注释  
		// 注释的内容
	  多行注释
		/*
		  注释的内容
		  注释的内容 可以有多行
		  /*
			错误：多行注释不能嵌套
		  */
		*/
		多行注释不能嵌套
	  文档注释  用来生成帮助文档
		/**
		  内容可以是一行到多行
		*/
  注释的特征：注释是在源代码中存在，给程序理解程序用，注释不会编译到字节码文件中。
