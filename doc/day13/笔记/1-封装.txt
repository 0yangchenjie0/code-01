面向对象编程有三大特性：封装、继承、多态。

封装：
    - 封装是面向对象编程的三大特征之一
    - 面向对象编程的第一步： 将属性和方法封装到一个类中
    - 外界使用类创建对象，然后 让对象调用方法和访问属性
    - 对象方法的细节都被封装在类的内部

    private 关键字
        - private 修饰构造方法，其它类无法创建该类的对象
        - private 修饰普通成员方法，该成员方法无法被其它类调用
        - private 修改成员变量，该成员变量无法被其它类访问
      增加了private 修饰符的成员变量，如何被外部类访问：
         以方法的方式暴露成员变量
      Java 类中成员变量一般使用private修饰，其它类访问该类的成员变量，
      通过该类提供的 getXXX()和 setXXX()方法来访问

    JavaBean
      JavaBean 是一个遵循特定写法的 Java 类，是一种 Java语言编写的可重用组件，它的方法命名，构造及行为必须符合特定的约定：
      1、这个类必须具有一个公共的 (public) 无参构造方法；
      2、所有属性私有化（private）；
      3、私有化的属性必须通过 public 类型的方法（getter和setter）暴露给其他程序，并且方法的命名也必须遵循一定的命名规范。
      4、这个类应是可序列化的。（后续讲解）
