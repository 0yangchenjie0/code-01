Java 基本语法
   1.注释
       单行、多行、文档
   2.标识符
     类、方法、方法的参数、变量起的名字
     由字母、数字、下划线、美元符号组成
     不能以数字开头
     不能使用关键字
   3.关键字
     用于特殊用途的英文单词
     全部小写
     public static void .....
   4.变量
     基本数据类型的变量或字符串可以使用以下方式定义：
        数据类型 变量名称 = 变量内容;
     引用数据类型
      数组：
        数据类型[] 变量名称 = new 数据类型[index];
      对象
        Random random = new Random();
   5.常量
     final  数据类型 变量名称；
